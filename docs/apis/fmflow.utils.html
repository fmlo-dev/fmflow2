fmflow.utils package¶

Module contents¶

Module for the internal use in FMFlow.

Users may not use this module directly. Developers must use this module like:

>>> from fmflow import utils as ut

map(func, sequence)[source]¶

Return a list of the results of applying the function to the sequence.

If self.mpcompatible is True, mapping is multiprocessed with the spacified number of processes (default is <CPU count> - 1). If False, mapping is singleprocessed (equivalent to the bulitin map function).

Parameters:
  • func (function) – Applying function.
  • sequence (list) – List of items to which function is applied.
